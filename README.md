# homebrew-tap

Homebrew tap for personal CLI tools. Formulae are published
automatically by [cargo-dist](https://github.com/axodotdev/cargo-dist).

## Usage

```sh
brew install ss0923/tap/<formula>
```

For example:

```sh
brew install ss0923/tap/limb
```

## Formulae

See [`Formula/`](./Formula) after the first release.
